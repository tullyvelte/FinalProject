Book Cover Judger

Version 1.0 - A Machine Learning Look at Book Cover Importance

Can you judge a book by its cover? Or how it might perform in Amazon reviews?

Let's see if we can use Machine Learning to find out.

We set out to create a ML model for the best and worst Amazon reviewed books and their covers. This site allows you to upload a book cover image, which is then processed and tagged through the GoogleVisionAPI, run through the model, and then receives predicted performance metrics based on previous Amazon reviews.


Upload up to 4 files and we'll make a prediction of its Amazon rating.
